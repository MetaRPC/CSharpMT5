    Get Position Stats By Symbol (GetPositionStatsBySymbolAsync)¬∂

    Portfolio method: Aggregate position statistics per symbol - count, volume, and P&L.

    API: MT5Service.GetPositionStatsBySymbolAsync(...) Region: [15] POSITION MONITORING

    Signature¬∂

    public static async Task<Dictionary<string, (int count, double totalVolume, double totalPnL)>>
    GetPositionStatsBySymbolAsync(
        this MT5Service svc,
        int timeoutSec = 20,
        CancellationToken ct = default)

    Returns¬∂

    Dictionary - Key: Symbol name (e.g., "EURUSD", "XAUUSD") - Value Tuple: - count - Number of open positions - totalVolume - Sum of all position volumes (lots) - totalPnL - Total profit/loss for this symbol

    Examples¬∂

var stats = await svc.GetPositionStatsBySymbolAsync();
