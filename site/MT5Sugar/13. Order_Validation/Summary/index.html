    Region 13: ORDER VALIDATION - Summary¶

    ⭐ CRITICAL REGION: Pre-flight checks for orders to prevent broker rejections and margin calls!

    Methods Overview¶

    1. ValidateOrderAsync() ⭐¶

    • Purpose: Full order validation before sending
    • Checks: Symbol availability, lot size, margin, stop levels
    • Returns: OrderCheckData with validation result + error codes
    • Usage: 
      var tradeReq = new MqlTradeRequest { Symbol = "EURUSD", Volume = 0.01, ... };
var checkReq = new OrderCheckRequest { MqlTradeRequest = tradeReq };
var result = await svc.ValidateOrderAsync(checkReq);

if (result.MqlTradeCheckResult.ReturnedCode == 0)
    Console.WriteLine("✅ Order valid");
else
    Console.WriteLine($"❌ Invalid: {result.MqlTradeCheckResult.Comment}");

    2. CalculateBuyMarginAsync()¶

    • Purpose: Calculate margin required for BUY order
    • Parameters: symbol, volume, price (0 = market)
    • Returns: Required margin amount in account currency
    • Usage: 
      double margin = await svc.CalculateBuyMarginAsync("EURUSD", volume: 0.10, price: 0);
Console.WriteLine($"Margin required: ${margin:F2}");

    3. CalculateSellMarginAsync()¶

    • Purpose: Calculate margin required for SELL order
    • Parameters: symbol, volume, price (0 = market)
    • Returns: Required margin amount in account currency
    • Usage: 
      double margin = await svc.CalculateSellMarginAsync("EURUSD", volume: 0.10, price: 0);
Console.WriteLine($"Margin required: ${margin:F2}");

    4. CheckMarginAvailabilityAsync() ⭐¶

    • Purpose: Check if account has enough free margin
    • Returns: (hasEnough, freeMargin, required) tuple
    • Usage: 
      var (hasEnough, free, required) = await svc.CheckMarginAvailabilityAsync(
    "EURUSD",
    volume: 1.0,
    isBuy: true);

if (hasEnough)
{
    Console.WriteLine($"✅ Can trade: ${free:F2} free vs ${required:F2} required");
    await svc.BuyMarketByRisk("EURUSD", stopPoints: 50, riskMoney: 100);
}
else
{
    Console.WriteLine($"❌ Insufficient margin: ${free:F2} < ${required:F2}");
}

    Real-World Example: Safe Order Placement¶

    public async Task<bool> PlaceBuyWithValidation(
    MT5Service svc,
    string symbol,
    double volume)
{
    // Step 1: Check margin availability
    var (hasMargin, free, required) = await svc.CheckMarginAvailabilityAsync(
        symbol,
        volume,
        isBuy: true);

    if (!hasMargin)
    {
        Console.WriteLine($"❌ Insufficient margin: ${free:F2} < ${required:F2}");
        return false;
    }

    Console.WriteLine($"✅ Margin OK: ${free:F2} available, ${required:F2} required");

    // Step 2: Build and validate order
    var tradeReq = new MqlTradeRequest
    {
        Symbol = symbol,
        Volume = volume,
        Type = ENUM_ORDER_TYPE.OrderTypeBuy,
        Price = 0  // Market price
    };

    var checkReq = new OrderCheckRequest { MqlTradeRequest = tradeReq };
    var validation = await svc.ValidateOrderAsync(checkReq);

    if (validation.MqlTradeCheckResult.ReturnedCode != 0)
    {
        Console.WriteLine($"❌ Validation failed: {validation.MqlTradeCheckResult.Comment}");
        return false;
    }

    Console.WriteLine("✅ Order validated successfully");

    // Step 3: Send order
    var result = await svc.BuyMarketAsync(symbol, volume);

    if (result.ReturnedCode == 10009)
    {
        Console.WriteLine($"✅ Order #{result.Order} executed");
        return true;
    }
    else
    {
        Console.WriteLine($"❌ Execution failed: {result.Comment}");
        return false;
    }
}

// Usage:
await PlaceBuyWithValidation(svc, "EURUSD", volume: 0.10);

    Why These Methods Matter¶

    Without validation: 

    // ❌ RISKY: Direct order without checks
await svc.BuyMarketAsync("EURUSD", 10.0);
// May fail if:
// - Insufficient margin
// - Symbol not available
// - Volume too large for account
// - Trading not allowed

    With validation: 

    // ✅ SAFE: Pre-flight checks
var (hasMargin, free, required) = await svc.CheckMarginAvailabilityAsync(
    "EURUSD", 10.0, isBuy: true);

if (hasMargin)
{
    var validation = await svc.ValidateOrderAsync(checkReq);
    if (validation.MqlTradeCheckResult.ReturnedCode == 0)
    {
        await svc.BuyMarketAsync("EURUSD", 10.0);  // Now safe!
    }
}

    Key Validation Checks¶

    Check Method What It Validates
    Margin CheckMarginAvailabilityAsync() Free margin >= required margin
    Buy Margin CalculateBuyMarginAsync() Exact margin for BUY order
    Sell Margin CalculateSellMarginAsync() Exact margin for SELL order
    Full Order ValidateOrderAsync() All parameters: symbol, volume, stops, margin

    Professional Pattern¶

    // ALWAYS validate before large orders
if (volume > 1.0)  // Large order threshold
{
    var (hasMargin, _, _) = await svc.CheckMarginAvailabilityAsync(
        symbol, volume, isBuy: true);

    if (!hasMargin)
    {
        Console.WriteLine("⚠️ Splitting order due to margin constraints");
        // Split into smaller orders...
        return;
    }
}

// Proceed with order
await svc.BuyMarketByRisk(symbol, stopPoints, riskMoney);

    Summary¶

    Region 13 (ORDER VALIDATION) prevents costly mistakes: * ✅ ValidateOrderAsync() - Full pre-flight check * ✅ CalculateBuyMarginAsync() - BUY margin calculation * ✅ CalculateSellMarginAsync() - SELL margin calculation * ✅ CheckMarginAvailabilityAsync() - Free margin check * ⭐ Use before ALL large orders to prevent margin calls!

    Professional traders ALWAYS validate before execution! 🛡️
